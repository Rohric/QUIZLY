# Quizzly

Quizzly lets you turn any YouTube video into an interactive quiz.
Paste a video URL, and the app automatically transcribes the audio and generates 10 questions using AI.

## Structure

| Folder | Description |
|--------|-------------|
| `backend/` | Django REST API — quiz generation, authentication, data models |
